<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the repository:</p>

```
git clone < repository-url >
```

<p>2. Navigate to the project directory:</p>

```
cd < project-directory >
```

<p>3. Install dependencies:</p>

```
 npm install
```

<p>4. Usage</p>

```
npm run dev
```
